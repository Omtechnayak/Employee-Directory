# Employee Directory

A simple and user-friendly **Employee Directory** built with **React**. Manage employee information easily with **add, view, update, delete, and search** functionalities. Perfect for practicing **CRUD operations** and building responsive interfaces.

## Features
- Add new employees  
- View all employees in a responsive table  
- Edit employee details  
- Delete employees  
- Search employees by name, department, or other fields  

## Tech Stack
- **Frontend:** React with Hooks  
- **Styling:** Bootstrap  
- **Routing:** React Router  
- **Backend:** JSON Server (mock API)
